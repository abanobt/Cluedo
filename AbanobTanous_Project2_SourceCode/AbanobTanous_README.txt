# Cluedo Game
This is a digital implementation of the classic murder mystery board game Cluedo (Clue).

## Source code
The entire source code is in the folder AbanobTanous_Project2_SourceCode/

## Requirements
Python 3.13
pygame 2.6
pygame_gui 0.6

## How to run
pip install pygame
pip install pygame_gui
python main.py
